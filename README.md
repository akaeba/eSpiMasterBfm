# eSpiMasterBfm
Enhanced SPI Master Bus Functional Model


## Features

The _eSpiMasterBfm_ provides VHDL procedures to interact with an eSPI endpoint. 
Currently are procedures available for:
 * IO Read/Write
 * Memory Read/Write
 * Endpoint Configuration
